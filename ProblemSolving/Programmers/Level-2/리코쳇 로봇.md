# 리코쳇 로봇

> https://school.programmers.co.kr/learn/courses/30/lessons/169199?language=java

### 내 코드

```java
import java.util.*;

class Solution {
    public int solution(String[] board) {
        int n = board.length;
        int m = board[0].length();
        char[][] map = new char[n][m];
        Pos pos = new Pos(0, 0, 0);
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                map[i][j] = board[i].charAt(j);
                if (map[i][j] == 'R') {
                    pos.r = i;
                    pos.c = j;
                    map[i][j] = 'V';
                }
            }
        }

        int[] dr = {0, 0, 1, -1};
        int[] dc = {1, -1, 0, 0};

        Queue<Pos> q = new ArrayDeque<>();
        q.offer(pos);
        while (!q.isEmpty()) {
            Pos cur = q.poll();
            if (map[cur.r][cur.c] == 'G') {
                return cur.cnt;
            }
            map[cur.r][cur.c] = 'V';

            for (int i = 0; i < 4; i++) {
                int nr = cur.r;
                int nc = cur.c;

                while (true) {
                    nr += dr[i];
                    nc += dc[i];
                    if (nr < 0 || nc < 0 || nr >= n || nc >= m || map[nr][nc] == 'D') {
                        nr -= dr[i];
                        nc -= dc[i];
                        break;
                    }
                }

                if (map[nr][nc] == '.' || map[nr][nc] == 'G') {
                    q.offer(new Pos(nr, nc, cur.cnt + 1));
                }
            }
        }
        return -1;
    }

    static class Pos {
        int r, c, cnt;

        public Pos(int r, int c, int cnt) {
            this.r = r;
            this.c = c;
            this.cnt = cnt;
        }
    }
}
```

### 피드백

- 목표로 하는 좌표까지 bfs 활용해서 도달하기. 최소한의 거리를 위해 bfs 사용.
